📝 Project Description
Glitch is a web-based hormonal health companion designed specifically for individuals with PCOS. Unlike traditional period trackers that assume regular 28-day cycles, our system detects irregularity patterns, calculates cycle variability, and generates intelligent insights based on user data.

🚨 The Problem Statement
Most existing period tracking applications assume predictable menstrual cycles. Individuals with PCOS experience irregular cycles, hormonal fluctuations, acne, cravings, and mood instability that are not accurately supported by conventional apps.
This leads to:
Poor cycle predictions
Lack of pattern detection
Emotional distress
No data-driven insight into irregularity

💡 The Solution
Glitch introduces a data-driven cycle intelligence dashboard that:
Calculates cycle lengths dynamically
Detects variability instead of assuming regularity
Generates stability classification
Tracks PCOS-specific symptoms
Produces rule-based hormonal insights
The system understands irregularity instead of penalizing it.

✨ Features
🔹 Feature 1: Period Logging
Users can log period start dates and store them locally.

🔹 Feature 2: Cycle Length Calculation
Automatically calculates cycle lengths between logged entries.

🔹 Feature 3: Irregularity Detection
Computes cycle variability and classifies stability as:
Stable
Moderate Irregularity
High Irregularity

🔹 Feature 4: Stability Meter
Dynamic progress bar that visually represents cycle consistency.

🔹 Feature 5: PCOS Symptom Tracker
Track:
Acne
Mood swings
Cravings
Hair fall
Bloating
Low energy

🔹 Feature 6: Insight Generator
Rule-based hormonal pattern analysis generates personalized health insights.

🔹 Feature 7: Data Visualization
Cycle lengths displayed using a dynamic Chart.js line graph.
